Buddy's capability probe — what the dock reads on mount: `{ ok, tools,
model, knowledge, threads }`. `model` is the alias Buddy runs on (`buddy`
when defined, else `default`), `knowledge` reports `semantic` once the boot
indexer has the handbook in mod-vectors (else `lexical`), and `threads` says
whether conversations persist (mod-store present). Privileged: gate the
trigger with the `admin` scope.
