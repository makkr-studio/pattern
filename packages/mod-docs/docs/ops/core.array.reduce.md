Folds a list to a single value by threading an accumulator through a **sub-workflow**: the sub-run receives `{ acc, item, index }` and returns `{ value }`, which becomes the next `acc` (seeded by `config.initial`). Reach for it when state must carry between elements; for an independent per-element transform use `core.array.map`. Unlike the other higher-order array ops it runs strictly sequentially (no `concurrency`), one linked sub-run per element.
