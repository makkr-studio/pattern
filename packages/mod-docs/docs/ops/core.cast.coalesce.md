The defaulting op — wire a primary value into `a` and fallbacks into `b`/`c`, and it emits the first that is neither null nor undefined (falling back to `null` if all are empty). Use it to seed a default without a branch; pair `c` with a `core.const.*` literal as the guaranteed last resort. Only null/undefined are skipped — `0`, `""`, and `false` count as present.
