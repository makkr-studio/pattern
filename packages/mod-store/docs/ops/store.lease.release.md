Release a lease you hold so the next run can claim it immediately instead of waiting out the TTL. You usually don't need it: a run auto-releases every lease it owns when it settles. Reach for explicit release when the owner ISN'T the runId — e.g. a streaming turn that owns `turn:{turnId}` and must drop the lock at its terminal event, because the run "settles" when its response stream is captured, not drained (see `store.lease.acquire`). Pass the same `key`/`owner` you acquired with.
