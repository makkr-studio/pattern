Release a lease you hold so the next run can claim it immediately, without waiting out the TTL. You usually don't need it: a run auto-releases every lease it owns when it settles. Reach for explicit release when the owner is something other than the runId, e.g. a streaming turn that owns `turn:{turnId}` and must drop the lock at its terminal event, because the run "settles" when its response stream is captured, before it drains (see `store.lease.acquire`). Pass the same `key`/`owner` you acquired with.
